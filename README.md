## Instalacion Web

Para instalar el proyecto es necesario hacer los siguientes comandos

```bash
  yarn install 
```


    
## Run Locally

Clone the project

```bash
  git clone https://github.com/MartinLoera/TechnicalTest
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn install 
  npm install
```

Start the server

```bash
  yarn dev 
  npm run dev
```


## Tech Stack

**Client:** React, MUI

**Server:** Node, Vite

**Mobile**: Ionic
